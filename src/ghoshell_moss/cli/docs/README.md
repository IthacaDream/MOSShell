# MOSS Documentation

Reference documentation for MOSS (Model-oriented Operating System Shell).

## Structure

- `en/` — English documentation (human readers)
- `zh/` — Chinese documentation (human readers)
- `ai/` — AI model documentation (high-density context for AI self-driving)

Run `moss docs` to see available doc sets. Use `moss --ai docs` for AI docs, `moss docs --lang zh` for Chinese.
